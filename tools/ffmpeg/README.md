# ffmpeg (LGPL build)

- 入手元: https://github.com/BtbN/FFmpeg-Builds/releases (`ffmpeg-master-latest-win64-lgpl.zip`)
- ライセンス: **LGPL 2.1+**(GPLコンポーネント非含有ビルド)。配布時はLGPL条項に従い、本アプリのライセンス表記画面に記載する
- 用途: 動画/音声からの音声トラック抽出と16kHz mono WAV変換のみ
- `ffmpeg.exe` / `ffprobe.exe` はサイズが大きいためgit管理外(.gitignore)。開発環境では上記URLから `win64-lgpl` を取得してこのフォルダに配置する
